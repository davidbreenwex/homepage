About Blingo

Metasearch

Blingo is a metasearch engine. Our aim is to take the results from the major search engines Google, Bing and Blekko and process them in such a way as to return a better results set than any of them. We use an aggregation technique known as Reciprocal Rank Fusion to aggregate the results. This algroithm has proved to be very effective in improving the accuracy of the results.

Boolean Search

The format for performing boolean searches is as follows:

AND is implied by default so there is no need to add it to your searches

OR searches can be performed in the following 2 ways:
"knives OR forks" (note uppercase OR)
"knives || forks"

NOT searches can be performed in the following 3 ways:
"knives NOT forks (note uppercase NOt
"knives - forks"
"knives -forks"

Query Preprocessing

When this option is turned on we do some work on your query. To improve performance we remove punctuation marks and stopwords from the query. Stopwords are taken from a list of very common words, e.g. "it" "the" or "for".

Clustering

When this option is turned on we categorise the Blingo results. To do this we use the k-means algorithm to group them into categories. Note: when this option is selected we will be a few seconds slower in returning results due to the extra processing required.
